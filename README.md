# 🔒 TinyVault

### Your Offline Secret Locker

A minimal retro pixel-inspired vault built with React Native & Expo for securely storing passwords, notes, API keys, WiFi credentials, and other sensitive information — completely offline.

## 🚀 Getting Started

```bash
git clone https://github.com/yourusername/TinyVault.git

cd TinyVault

npm install

npx expo start
```
